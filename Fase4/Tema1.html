Hojas de Estilo en Cascada CSS


La sigla CSS corresponde a la expresión inglesa Cascading StyleSheets, que puede traducirse como “Hojas de estilo en cascada”. El concepto se utiliza en el ámbito de la informática para referirse a un lenguaje empleado en el diseño gráfico.

Es un lenguaje de hojas de estilo. CSS nos permiten definir la apariencia de cada elemento dentro de los documentos en HTML.

El CSS son las hojas de estilo en cascada que definen la apariencia de nuestros documentos en HTML. Para que nuestros estilos CSS se apliquen correctamente a nuestras páginas web, debemos utilizar la etiqueta link con el atributo href y la ruta a nuestro archivo .css

Los Selectores nos permiten conectar las etiquetas de HTML con sus respectivos estilos en CSS. Existen muchos tipos de selectores, por ejemplo, los selectores de clase buscan el contenido que tenga un cierto valor en su atributo class.

También tenemos selectores de tipo ID (estos selectores solo pueden aplicar a un elemento, no va a funcionar si escribimos dos o más etiquetas con el mismo ID En CSS utilizamos atributos para definir los estilos de nuestros elementos, podemos modificar el color de la letra, tamaño, color de fondo, espaciado, entre otras.
