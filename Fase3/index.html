Propósito del sitio


El propósito de este sitio web, es brindar información y recursos educativos digitales para el aprendizaje, sobre las etiquetas básicas del HTML, atributos y sus valores para la construcción de sitios web básicos y luego crear aplicaciones web interactivas, que puedan ayudar a la implementación de herramientas educativas, para suplir la necesidad de la presencialidad en recintos eduactivos, todo esto a causa del COVID-19.

Etiquetas Básicas de HTML

Atributos y valores


Los atributos en HTML se pueden considerar como información adicional que se coloca siempre en la etiqueta de apertura, esto para añadir, complementar información o significado al elemento HTML al que estamos colocando.
